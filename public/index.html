FilterBar Component

I created this component to handle filters values and compose more complex components.

It expose a callback function that allow to execute an a action (post a form, create a xhr request, etc).

It support the UIFilterDropdown, UIFilterDropdownAsync and UISearch components.

Usage:

const props = {
filters: [{
filterType: 'UIFilterDropdown',
title: 'Status',
name: 'active',
multiple: false,
opts: [{
label: 'Active',
value: 1
}, {
label: 'Inactive',
value: 0
}]
],
lockOnChange: false,
callback: function(filters, filter) {}
}

ReactDOM.render(<UIFilterBar {...props} />, document.getElementById('root'));
